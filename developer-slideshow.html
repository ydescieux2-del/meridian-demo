What You Built

MERIDIAN: Unified operations dashboard for Nicole Feenstra (DNA Agency)

  • Single-page application (SPA) built with vanilla JavaScript
  • Hosted on GitHub Pages (Phase 1), upgrade to Vercel/Railway (Phase 2)
  • Browser localStorage (Phase 1), PostgreSQL (Phase 2)
  • Git push to production — zero downtime deployments
  • URL-only access — no installation or authentication required
Key Advantage: You control the entire codebase; updates deploy instantly without touching Nicole's computer

Account Setup: Plaid (Banks & Credit Cards)

Your Setup (~2 hours)

  1. Go to plaid.com → create developer account
  2. Get Client ID + Secret
  3. Implement Plaid Link on frontend
  4. Backend: Exchange public token for permanent access token
  5. Implement 5-min polling + token refresh logic

Nicole's Setup

Clicks "Connect Account" → Plaid window → searches bank → logs in directly → selects accounts → done

Cost: ~$1–2/mo (2–3 accounts)

Account Setup: QuickBooks

Your Setup (~4 hours)

  1. Go to developer.intuit.com → create app
  2. Get Client ID, Client Secret, Redirect URI
  3. Implement OAuth 2.0 flow
  4. Store tokens securely in backend (encrypted)
  5. Implement token refresh (QB tokens expire in 60 days)
  6. Periodic API calls for P&L, invoices, payroll

Nicole's Setup

Clicks "Connect QuickBooks" → logs into QB → approves permissions → connection live

Cost: Free (included in her QB subscription)

Security Considerations

API Key Management

  • Never store API keys in frontend code
  • Always proxy API calls through backend
  • Rotate keys quarterly
  • Use environment variables (never hardcode)

Phase 2 Data Security

  • Database encryption for sensitive fields
  • HTTPS only (no HTTP)
  • Audit logs for all data access

OAuth Best Practices

  • State parameter to prevent CSRF attacks
  • Scope minimization (only request needed permissions)
  • Token expiration + refresh logic

Common Developer Questions

Q: How do I test locally?
A: python3 -m http.server 8000 then open localhost:8000

Q: Can I use a custom domain?
A: Yes. Repo Settings → Pages → Custom domain. Add CNAME record to registrar.

Q: What if localStorage quota is exceeded?
A: Safari ~50MB, Chrome ~10MB. If 500+ contacts, implement Phase 2 or aggressive archiving.

Q: Can Nicole use MERIDIAN on multiple devices in Phase 1?
A: Not recommended. Data won't sync. Use Phase 2 for multi-device.
