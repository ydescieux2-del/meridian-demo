What MERIDIAN Is

A private, branded operations dashboard built specifically for Nicole Feenstra.

Four Core Functions:

  • Campaign Command Center — Track all active ad campaigns
  • CRM & Pipeline — Full contact management
  • Nonprofit Finance — Donor pipeline & grant tracker
  • Creative Studio — AI-powered deck generation

Built on a single index.html file · No app installations · No login required

Current Limitation

Browser LocalStorage

All data (campaigns, contacts, donors, grants) is stored in browser localStorage.

What This Means:

Data lives in whichever browser on whichever device Nicole uses. If she switches browsers or devices, it starts fresh.

Guidance: Always open MERIDIAN from the same browser on the same device until Phase 2 upgrades to a real database backend.

Connecting Real Financial Data

QuickBooks Online

What it gives: Live income/expense data, invoices, cash flow, payroll

Nicole does: Authorize through a standard login popup (no credentials shared)

You do: Register a developer app at developer.intuit.com (~4–6 hrs Phase 2)

Cost: Free (included with existing QB subscription)

Connecting Real Financial Data

Bank Accounts & Credit Cards (Plaid)

What it gives: Live transactions from 12,000+ institutions

Security: Nicole logs in directly with her bank (you never see credentials)

Nicole does: Click "Connect Account" and authorize through Plaid Link

You do: Create free Plaid developer account, add Plaid Link to frontend (~3–5 hrs Phase 2)

Cost: ~$0–$15/mo (See Section 4)

What Nicole Provides (The Short List)

She provides NOTHING sensitive.

  • ✓ QuickBooks company ID
  • ✓ Confirmation she's authorized QB connection
  • ✓ Confirmation she's connected her bank via Plaid
  • ✓ Preferred domain name (e.g., meridian.dnaagency.com)

She NEVER gives: Bank logins, passwords, credit card numbers, SSNs

The Data Flow

How everything connects:

  • Finance Tab: QuickBooks + Plaid → Live totals, pipeline, drill-down
  • Campaign Tab: Meta API + Google Ads API → Live spend, leads, CPL
  • CRM Tab: Manual + CSV import → Pipeline, edit, search
  • Creative Studio: MERIDIAN AI + Gamma + Canva → On-demand decks

All feeding into one elegant dashboard

Why Monthly Maintenance Is Essential

What It Covers

  • Data Integrity: API tokens expire, connections drift
  • Platform Updates: Meta, Google, QuickBooks APIs change constantly
  • Security Monitoring: Key rotation, access logs, backups
  • Campaign Accuracy: Spot-check that numbers match ad platforms
  • Feature Deployment: New capabilities added on rolling basis

What Monthly Oversight Provides

  • Strategic Interpretation — What do the numbers mean? What should Nicole do?
  • Priority Calls — When three things need attention, what's urgent?
  • Proactive Catch — See problems before they happen
  • Institutional Memory — History of every campaign, donor, grant inside the platform
  • Single Point of Contact — One person owns the entire stack

The Cost of NOT Having It

❌ Broken API connection = stale financial data (no one notices)

❌ Campaign overspends (pacing problem not caught)

❌ Donor goes cold (wrong contact date in CRM)

❌ Grant deadline missed (no one watching tracker)

❌ Platform becomes shelfware within 90 days

MERIDIAN is infrastructure.

Infrastructure without maintenance fails.
